HPX provides a comprehensive implementation of parallel algorithms that serves as both a standards-compliant C++ library and a high-performance computing runtime system. However, parallel algorithms based on the partitioner_with_cleanup component currently don’t have support for the Senders/Receivers programming model, which will be standardized as part of the C++26 execution library.
